Créer une clé API¶

Pour créer la clé API il suffit de se connecter sur: https://openrouteservice.org Il faut cliquer sur login et créer un compte (avec github si vous travaillez dessus) puis sur Request a token remplissez les cases en nommant votre clé. Votre clé est créée il vous suffit de la copier et de la mettre dans le code.
