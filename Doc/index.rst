Bienvenue sur la documentation de Coberny
=================================================
Utilisez Coberny pour visualiser, optimiser et économiser votre temps et votre argent lors de vos déplacements en Occitanie.
Ce package est destiné à visualiser l'autoroute par laquelle vous souhaitez passer ainsi qu'une distribution des prix sur les portions. Il est également destiné à déterminer le trajet le moins coûteux  avec un nombre de sorties maximum imposées.
